# Handoff

Short-term working context lives here.

## Files

- `CURRENT_CONTEXT.md`: generated by `handoff`; ask before loading it on `run` / `load`.
- `WORKFLOW_SCRATCH.md`: temporary workspace for active workflows.
- `Archive/`: completed scratch files.

## Rules

- `CURRENT_CONTEXT.md` is not long-term memory.
- `clear-handoff` deletes only `CURRENT_CONTEXT.md`.
- Long-term decisions should go through `crystallize`.
- Only one active `WORKFLOW_SCRATCH.md` should exist at a time.
