# Tutorial 02: Add Research Pack

## Goal

Add source-backed research workflows.

## Steps

1. Install Core first.
2. Copy everything inside `templates/packs/research/` into the vault root.
3. Ask your LLM:

```txt
StandardResearch: compare local markdown knowledge bases and vector RAG for a small team.
```

4. Ask the agent to preserve source URLs.
5. Review the output before saving any note.

## Expected Result

You should get:

- existing-vault coverage
- external source findings
- synthesis
- review notes
- a proposed source-backed note
